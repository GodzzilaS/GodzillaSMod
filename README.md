# Вспомогательный мод для cristalix MiniGames.

Добавляет:

- Время в чате.


- Команды:
    - /gtime - Скрыть время.
    - /gtimecolor #hex - Поменять цвет в чате.
    - /glist - Вывести список людей на сервере.
    - /gsetrpc [param] - Поставить [param] в качестве статуса в discord RPC.
